JavaScript shartli operatorlar bo'yicha qo'llanma

1. If-else

if va else shartli operatorlari shartlarni tekshirish uchun ishlatiladi.

1.1 If

Agar shart bajarilsa, kod bajariladi.

        
let age = 20;

if (age >= 18) {
    console.log("Siz katta yoshdasiz.");
}

1.2 If-else

Agar shart bajarilsa, birinchi kod bajariladi, aks holda ikkinchi kod bajariladi.

        
let age = 16;

if (age >= 18) {
    console.log("Siz katta yoshdasiz.");
} else {
    console.log("Siz hali katta yoshda emassiz.");
}

1.3 If-else if-else

Birinchi shart bajarilmasa, keyingi shartlar tekshiriladi va oxirgi else qismi bajariladi.

        
let age = 65;

if (age < 18) {
    console.log("Siz hali katta yoshda emassiz.");
} else if (age >= 18 && age < 65) {
    console.log("Siz katta yoshdasiz.");
} else {
    console.log("Siz pensiya yoshidasiz.");
}

2. Switch-case

switch va case operatorlari ko'p shartlarni tekshirish uchun ishlatiladi.

2.1 Switch-case

O'zgaruvchi qiymatiga qarab, tegishli case bajariladi.

        
let fruit = "olma";

switch (fruit) {
    case "olma":
        console.log("Bu olma.");
        break;
    case "banan":
        console.log("Bu banan.");
        break;
    case "gilos":
        console.log("Bu gilos.");
        break;
    default:
        console.log("Bunday meva yo'q.");
}

2.2 Multiple cases

Bir nechta case bloklari bir xil kodni bajarishi mumkin.

        
let day = 2;

switch (day) {
    case 1:
    case 2:
    case 3:
    case 4:
    case 5:
        console.log("Bu ish kuni.");
        break;
    case 6:
    case 7:
        console.log("Bu dam olish kuni.");
        break;
    default:
        console.log("Noto'g'ri kun.");
}
